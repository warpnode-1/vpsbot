# vps Deploy Bot commands 
**Must have subscribed to yash**
**1st Command** 
```
sudo apt update
```
**2nd Command** 
```
sudo apt install git -y
```
**3rd Command** 
```
git clone https://github.com/Ayuxd-yt/vpsdeploybot
```
**4th Command** 
```
sudo apt install python3-pip -y
```
**5th Command** 
```
cd vpsdeploybot
```
**6th Command** 
```
pip install -r requirements.txt
```
**7th Command** 
```
touch Dockerfile
```
**8th Command** 
```
docker build -t ubuntu-22.04-with-tmate .
```
**9th Command** 
```
nano bot.py
```
**add token!**
**10th Command** 
```
python3 bot.py
```
**add token!**
